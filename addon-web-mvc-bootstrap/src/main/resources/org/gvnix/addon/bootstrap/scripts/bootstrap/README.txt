Offcanvas.js
------------

Js necessary to build a toggleable off-canvas navigation menu for use with Bootstrap.
Demo in http://acmetech.github.io/todc-bootstrap-3/examples/offcanvas/
In use in ../webapp/WEB-INF/layouts/default-menu-cols.jspx

Bootstrap html5 Integration
---------------------------

For IE recognizes html5 tags.

* html5shiv.js: minified version
* License:  MIT/GPL2 Licensed
* Downloaded with Bootstrap

Bootstrap media queries Integration
-----------------------------------

For min/max-width CSS3 Media Queries (for IE 6-8, and more)

* html5shiv.js: minified version
* License:  Dual MIT/BSD license
* Downloaded with Bootstrap


Bootstrap Datatables Integration
--------------------------------

Integration classes Bootstrap in Datatables

* dataTables.bootstrap.js: uncompressed version
* Downloaded from: https://github.com/DataTables/Plugins/tree/master/integration/bootstrap
* Related with css styles in styles/datatables/dataTables.bootstrap.css

IMPORTANT: compare this version with new library updates, there are customized properties made by DISID.

NOTE

By default the addon will use the compressed versions, change the reference
to the uncompressed ones in the "load-scripts-footer.tagx" if needed.
